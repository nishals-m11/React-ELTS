# Education Loan Tracking System

A modern frontend web application built using React for managing education loan applications, repayment tracking, and student loan monitoring. The platform provides role-based access for students and administrators with an interactive and responsive user interface.

## 🚀 Features
- 🔐 Role-based Login System.
- 📝 Education Loan Application Form.
- 📊 Student Loan Dashboard.
- 💳 Loan Repayment Tracking.
- ⏰ EMI Reminder Notifications.
- 📱 Responsive User Interface.
- ⚡ Single Page Application (SPA).
- 🔄 Dynamic Routing using React Router.
- 🧠 State Management using Context API & useReducer.

## 🛠️ Tech Stack

| Technology       | Purpose             |
| ---------------- | ------------------- |
| React            | Frontend Framework  |
| React Router DOM | Client-side Routing |
| Context API      | State Management    |
| useReducer       | State Handling      |
| Vite             | Build Tool          |
| React Icons      | UI Icons            |
| CSS              | Styling             |

The application follows a component-based architecture with reusable React components and efficient state management.

## ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/nishals-m11/React-ELTS.git
```

### Navigate to Project Directory

```bash
cd React-ELTS/reactapp
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```
